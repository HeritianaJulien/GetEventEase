# GETEventEase

GETEventEase est une application mobile conçue pour la mention Télécommunication. Elle centralise les informations nécessaires pour les événements de l'année universitaire : conférences, ateliers, réunions, soutenances et annonces pédagogiques.

## Objectif

- Centraliser les informations des événements de la filière Télécommunication.
- Fournir un calendrier annuel des événements.
- Permettre l'accès aux détails de chaque événement (date, heure, lieu, intervenant).
- Envoyer des notifications ciblées aux étudiants et enseignants.
- Préparer le déploiement officiel en tant que version interne / pré-release.

## Fonctionnalités principales

- Consultation du calendrier des événements.
- Fiches détaillées d'événement.
- Gestion des inscriptions / confirmations.
- Notifications pour les nouveaux événements et changements.
- Accès aux ressources associées (liens de diffusion, documents, contacts organisateurs).
- Interface mobile simple et adaptée aux besoins du département Télécommunication.

## Compatibilité Android

- Minimum Android 9.0 (API 28)
- Cible Android 13 (API 36)
- `versionCode`: 1
- `versionName`: 1.0

> Note : l'application n'est pas encore déployée officiellement. Cette version sert de pré-release interne pour la mention Télécommunication.

## Release et APK

- Nom recommandé pour l'APK : `GETEventEase-1.0.0-telecom.apk`
- Tag Git recommandé : `v1.0.0-telecom` ou `v1.0.0-telecom-beta.1` si vous marquez une pré-release.

## Installation

1. Télécharger l'APK depuis les releases GitHub.
2. Autoriser l'installation d'applications provenant de sources externes sur l'appareil Android.
3. Installer l'APK.
